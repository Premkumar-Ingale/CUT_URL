# Audio Assets

Place your audio files here. Supported formats: `.mp3`, `.wav`, `.ogg`

## Expected files

| Filename | Usage |
|---|---|
| `hover.mp3` | Plays on tech badge hover in the About page |
| `bg_music.mp3` | Background music (reserved for future use) |

## How to add audio

1. Drop your `.mp3` file into this folder with the correct filename above.
2. Open `src/assets/index.js` and uncomment the relevant import line.
3. The component will automatically pick it up — no other changes needed.
